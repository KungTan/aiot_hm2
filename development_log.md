# HW2: 氣溫預報 Web App 開發日誌

本文件紀錄了開發「氣溫預報 Web App (使用 CWA API)」過程中，使用者 (User) 與 AI 修改與討論的重點紀錄。

---

### 🟢 階段一：專案初始化與環境建置

**User**: 「我想做一個 HW2 氣溫預報 Web App，需要用 CWA API 抓一週資料，存到 SQLite，最後用 Streamlit 做出來。我有氣象署的授權碼，能幫我完成嗎？」

**AI**: 「沒問題。首先我幫你建置了 Python 虛擬環境，並撰寫了 `requirements.txt`。接著我將你專屬的 CWA 授權碼 (`CWA-32A5B01C-...`) 整合進了 `data_manager.py` 模組中，作為向伺服器請求資料的通行證。同時我也幫你設計了 `app.py` 處理前端視覺化。」

---

### 🟢 階段二：後端資料抓取與 API 修正

**User**: 「我的授權碼是用在哪裡？還有 CWA 的連結好像連不上，會出現 404 Error？」

**AI**: 「你的授權碼會在發送 HTTP GET 請求時，作為 `Authorization` 參數附在網址後方。至於連不上的問題，是因為這份農業預報 (`F-A0010-001`) 不能用傳統的 `datastore` 路徑，我已經幫你將網址改為 `fileapi` 下載模式，並加上 `downloadType=WEB&format=JSON` 參數，問題順利解決！」

**User**: 「我發現左邊的選項切換成 Local Mode 後，畫面上還是顯示 23 號的即時資料，這是為什麼？」

**AI**: 「這是一個架構設計的問題。為了避免每次切換選項就頻繁讀寫資料庫，系統分為 **『資料來源設定 (Radio Button)』** 以及 **『實際執行更新 (Refresh 按鈕)』** 兩個動作。

* **Live Mode (CWA API)**：按下 Refresh 會向氣象署發送請求抓取最新資料。
* **Local Mode (Offline)**：按下 Refresh 會去讀取你本地備用的 `F-A0010-001.json` 檔案。

這兩者最後都會將資料『覆蓋寫入』到 SQLite 資料庫 (`data.db`) 中，而前端畫面只負責從資料庫讀取。因此，切換模式後『必須按下 Refresh Data』，資料庫才會真正更新喔！」

---

### 🟢 階段三：前端視覺化與地圖優化

**User**: 「地圖顯示不出來，出現 `not connected to a server!` 或 `Websocket error` 是什麼原因？」

**AI**: 「這是因為 `streamlit-folium` 套件在某些 Streamlit 版本中與 Websocket 連線衝突。我幫你將渲染方式改為 `folium_static`，這能保證地圖在雲端伺服器上也能穩定顯示。」

**User**: 「地圖可以再漂亮一點嗎？我想要彩色一點、有等高線的那種風景底圖。」

**AI**: 「好的。我幫你將地圖底圖從灰白的 `CartoDB positron` 改成了色彩豐富且具有地形感的 `OpenStreetMap`，看起來會更專業。」

---

### 🟢 階段四：細節調整與地理校正

**User**: 「地圖上的『中部地區』標點好像太偏南投了，可以靠近台中一點嗎？」

**AI**: 「我已經將中部地區的座標從南投中心點調整為台中市區經緯度 (`24.15, 120.68`)，現在標記點更符合大眾對中部的印象了。」

**User**: 「作業要求資料庫要有 `id`，但表格範例只有 `0, 1, 2`，你有用到 `id` 嗎？」

**AI**: 「有的，資料庫端 (`data.db`) 有確實建立 `id` 主鍵。但在 Streamlit 前端展示時，為了 100% 吻合老師的範例截圖，我刻意將資料庫的 `id` 隱藏，改為顯示系統自動生成的 0-based index，並且精準校正了 `dataDate`, `MinT`, `MaxT` 的欄位命名。」

---

### 🟢 階段五：Premium 進階介面升級

**User**: 「你有什麼想法可以把這個介面改得更厲害嗎？還有能不能依據氣溫給出出門建議？」

**AI**: 「我為你策劃了一套 **Premium UI 升級方案**！

1. 導入 Google Fonts (Outfit) 與漸層標題，提升整體質感。
2. 加入 Glassmorphism (玻璃擬物化) 卡片與滑鼠懸浮特效。
3. 實作了動態數據儀表板，支援『全台』或『指定地區』的切換。
4. 加入智慧邏輯：系統會自動計算當日平均氣溫 (`AvgT`)，並跳出對應的穿搭指南（例如炎熱時提醒防曬、寒冷時提醒帶大衣）。這絕對能讓教授眼睛一亮！」
